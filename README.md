# Project 5 — CV Layout 5

A magazine-style, print-friendly **one-page CV** built with **HTML & CSS**, featuring a two-column layout inspired by newspaper design.

## ✨ Sections

- About  
- Experience  
- Skills  
- Projects  
- Education  

## 🚀 Features

- Two-column “newspaper” layout  
- CSS-only interactions (`:has()` for active states)  
- Masonry-style project list using CSS columns  
- Auto dark mode & responsive design  
- Print-friendly layout for clean PDF export
